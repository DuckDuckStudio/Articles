# 生物复习随便写写

| 类型 | 定义 | 例如 |
|-----|-----|-----|
| 特异性免疫 | 只对某一种 | 疫苗 |
| 非特异性免疫 | 对多种 | 生来就有的 |

---

细胞核、质、膜、线粒体 -> 都有  
叶绿体 -> **绿色**植物  

叶绿体 -> 光合  
线粒体 -> 呼吸  

---

动脉 ≠ 动脉血  

e.g.  
肺动脉 -> 静脉血  
肺静脉 -> 动脉血  

动脉: 出去的  
静脉: 回去的  

---

心脏房室间 -> 房室瓣  
动脉 -> 动脉瓣  

Keep 单向流
