# 部分Microsoft Visual Code的指令  
运行指令 `Ctrl` + `Shift` + `P`  
多行末尾添加 `Shift` + `Alt` + `i`  
关闭文件夹 `Ctrl` + `K` 先按，然后在加上按 `F`  
添加/删除行前空格 `选中多行`:
- `Shift` + `Tab` -> 删除 4 个空格
- `Tab` -> 4 个空格

折叠整个文件中的代码块: (`Ctrl`+`K`)+(`Ctrl`+`0`)  
展开整个文件中的代码: (`Ctrl`+`K`)+(`Ctrl`+`J`)  

关闭所有选项卡: `Ctrl`+`K`+`W`
