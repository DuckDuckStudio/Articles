# 我该去哪里听电台 / 广播

你可以通过在线收听或收听对应频率来听广播，以下是相关信息介绍。  

## 在线收听

> 有些台会选择第三方平台作为官方收听途径，常见的有微信、云听、喜马拉雅、蜻蜓 FM。
