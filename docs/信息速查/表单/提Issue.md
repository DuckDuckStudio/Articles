# 在提 Issue 前，请先问自己几个问题

1. 我知道如何最小复现这个问题吗？
2. 这个问题能稳定复现吗？
   1. 在不同的网络环境下能否复现？
   2. 在不同的设备上能否复现？
   3. 是不是时而能复现、时而如预期工作？
3. 确定这是一个问题而不是故意这样做的吗？
4. 别人有没有提出过相同或类似的问题？
5. 我真的知道这个问题是什么导致的吗？
   - 如果不知道确切成因，是否有可能性较高的猜测？
6. 我能清楚的用项目维护者使用的语言描述问题吗？
   1. 描述中有没有存在可能有歧义的内容？
      - 如 it、that、this 等是否能一眼看出指代的东西？
   2. 描述是否存在语法错误？
   3. 描述是否通顺？
7. 我能提供复现过程的截图 / 视频 / 日志吗？
   1. 我提供的截图有包含关键信息吗？
   2. 我提供的日志能否体现问题的成因 / 问题的出现 / 其他关键信息？
      - 如果不能，则没有必要提供日志
8. 我熟悉这个项目吗？
   - 如果不熟悉，则无法准确的和维护者沟通。
9. 项目近一年内有新提交吗？
   - 如果没有，则代表该项目可能已经被“遗弃”
10. 我知道怎么解决这个问题吗？
   - 大概估计解决这个问题的复杂程度，以及是否有必要进行修复。如果解决方案复杂且没什么修复必要，问题很可能会变成特性。
