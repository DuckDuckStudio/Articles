# 限制挂载和互操作

出于安全性考虑，禁止挂载 Windows 文件到 `/mnt/` 下，并禁止互操作。  
部分互操作（如 `code`）需要先挂载 Windows 文件。

## 方法

wsl.conf 文档: https://learn.microsoft.com/zh-cn/windows/wsl/wsl-config#wslconf

打开 wsl.conf:  
```shell
sudo nano /etc/wsl.conf
```

添加这些配置:  
```conf
# 不挂载 Windows 文件
[automount]
enabled=false

# 不与 Windows 互操作
[interop]
enabled=false
appendWindowsPath=false
```
